Here are the steps to get Cordova running on OSX Yosemite (and others.)
Prerequisites:
  • OSX 10.6 or later.
  • NodeJS and NPM
  • Brew
  1. Download Android SDK [link] (not android studio) and unpack in location of choice (/Applications/Android/sdk)

    If you have had the SDK previously installed, you should run /Applications/Android/sdk/tools/android and install any updates that are available.
  2. Add SDK location to
    ~/.bash_profile
    file for Android Building:

    Add the following lines to your

    ~/.bash_profile
    file:

    [shell] export ANDROID_HOME=/Applications/Android/sdk export PATH=$PATH:$ANDROID_HOME:$ANDROID_HOME/platform-tools:$ANDROID_HOME/tools [/shell]

    Then type

    source ~/.bash_profile

  3. Install Apache Ant:

    [shell] brew update brew install ant [/shell]
